# [`node-llama-cpp`](https://github.com/isdk/llama-node)
This is an extension of the [`@isdk/llama-node-linux-x64-cuda`](https://www.npmjs.com/package/@isdk/llama-node-linux-x64-cuda) package
that provides a prebuilt binary for [`node-llama-cpp`](https://github.com/isdk/llama-node) for Linux x64 with CUDA support.

Do not install this package directly.
