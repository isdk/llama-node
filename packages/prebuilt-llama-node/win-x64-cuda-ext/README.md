# [`@isdk/llama-node`](https://github.com/isdk/llama-node)
This is an extension of the [`@isdk/llama-node-win-x64-cuda`](https://www.npmjs.com/package/@isdk/llama-node-win-x64-cuda) package
that provides a prebuilt binary for [`@isdk/llama-node`](https://github.com/isdk/llama-node) for Windows x64 with CUDA support.

Do not install this package directly.
