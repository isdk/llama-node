# @isdk/llama-node Templates

Example templates demonstrating how to use `@isdk/llama-node` in different environments.

## Available Templates

- **node-typescript**: Basic Node.js + TypeScript setup with low-level llama.cpp bindings
- **electron-typescript-react**: Electron app with React frontend using llama.cpp in the main process

These templates showcase the core functionality of `@isdk/llama-node`:
- Model loading and context management
- Text completion with low-level APIs
- Grammar-based generation
- Tokenization and embeddings
