# Node + TypeScript + `@isdk/llama-node`

This template provides a minimal setup to get Node working with TypeScript and `@isdk/llama-node`, demonstrating low-level llama.cpp bindings usage.

## Features

- Low-level text completion API
- Grammar-based JSON generation
- Tokenization examples
- ESLint configuration

## Get started

Install node modules:
```bash
npm install
```

Start the project:
```bash
npm start
```

## What's Included

This template demonstrates:
- Model loading and context creation
- Text completion with `LlamaCompletion`
- JSON Schema grammar for structured output
- Token-level control and streaming
